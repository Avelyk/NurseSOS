# NurseSOS

NurseSOS is a technology-driven healthcare platform designed to provide immediate nursing assistance, health consultations, and emergency response coordination through a responsive web and mobile-friendly application. 

This project is a functional frontend prototype built as part of a business plan proposal to address healthcare accessibility gaps in the Philippines.

---

## 🛠️ Tech Stack & Structure

This prototype is kept clean, light, and beginner-friendly so it can be easily run and modified:

* **HTML5** – Core structure and layout of the platform.
* **CSS3** – Custom styling with a calming medical theme, using Flexbox/Grid for a fully responsive mobile and desktop view.
* **JavaScript (Vanilla)** – Interactive front-end logic and simulated live chat functionality.
* **Node.js** – A simple, lightweight server to run the app locally.

Link to website: https://avelyk.github.io/NurseSOS/
